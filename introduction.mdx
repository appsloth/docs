## What is Appsloth?

Appsloth is a collection of ready-to-use Next.js components designed to enhance user engagement in your application. Built on top of shadcn's beautiful component library, it provides a suite of customizable components for collecting feedback, managing communications, and improving user retention. Each component automatically collects and sends data to your Appsloth dashboard, eliminating the need to build and maintain your own data infrastructure.

Our key features include:

- [Banners](/features/banners) - Create eye-catching announcement bars to highlight important updates and promotions
- [Feedback](/features/feedback) - Collect and manage user feedback to continuously improve your product
- [Inbox](/features/inbox) - Send targeted in-app messages and notifications to engage with your users
- [Retention Modal](/features/retention) - Reduce churn with smart cancellation flows and targeted offers
- [Status Page](/features/status) - Keep users informed about your service availability and incidents
- [Testimonials](/features/testimonials) - Showcase user success stories and build social proof

## Who is this for?

Appsloth is perfect for React and Next.js developers who want to:

- Add polished, production-ready engagement features without building from scratch
- Use beautiful, shadcn-based components while maintaining full customization control
- Skip building complex data infrastructure for user engagement features
- Save development time with drop-in solutions for common user engagement patterns
- Maintain a consistent look and feel with their Next.js application
- Get instant access to analytics and insights through the Appsloth dashboard
